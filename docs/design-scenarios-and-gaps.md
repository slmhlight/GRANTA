# 기계 설계 엔지니어 시나리오 & 기능 점검

> AM Materials Explorer가 실제 기계 설계 엔지니어의 재료 선택 업무를 얼마나 지원하는지
> 점검하고, 개선점을 우선순위별로 정리한 문서. (작성: 기능 점검 라운드)

## 0. 대상 사용자 & 일반 워크플로우

전형적인 Ashby 재료 선택법 + 실무 검증 흐름:

1. **제약(constraints)** — 반드시 만족해야 하는 한계 (예: σy ≥ 800 MPa, 공정 = LPBF, 최대사용온도 ≥ 600 °C)
2. **목적함수(material index)** — 최대/최소화할 성능 지표 (예: 경량 강성 E/ρ)
3. **후보 비교(compare)** — 상위 후보를 물성별로 나란히 비교
4. **검증(verify)** — 데이터시트/규격/출처로 확정, 조건(열처리·방향·온도) 확인

현재 앱은 이 4단계를 모두 일부 이상 지원한다. 아래는 시나리오별 상세 점검.

범례: ✅ 지원 · 🟡 부분 지원/우회 가능 · ⚠️ 미지원(개선 필요)

## 1. 시나리오별 점검

### S1. 경량 구조 브래킷 (LPBF 출력)
질량 최소, 강성/강도 제약, 금속 AM 출력 가능.
- ✅ 인덱스 `E/ρ`, `σy/ρ`, 경량 보·판 `E½/ρ`·`E⅓/ρ`·`σy⅔/ρ` 제공
- ✅ 공정 필터(LPBF/DMLS/EBM 등), 밀도·항복·탄성계수 범위 필터
- ✅ 다중 인덱스 제약(강성+강도 동시), 후보 Compare, CSV 내보내기
- 🟡 피로(`fatigue_strength`)는 필터·비교 가능하나 단일값(끝점)이며 "est." 라벨
- ⚠️ **빌드 방향 이방성(XY vs Z)** 이 정량 비교 불가 — 상세에 "방향에 따라 다름" 텍스트 노트만
- ⚠️ 출력성 제약(최소 벽두께, 오버행/서포트, 빌드 부피)·표면조도(as-built) 없음

### S2. 고온 부품 (터빈 디스크 / 배기 / 매니폴드)
600–900 °C 강도 유지, 산화/부식, 피로·크리프.
- ✅ `max_service_temp` 필터, 내식성(정성) 필터
- ✅ 일부 합금 **온도별 항복/인장(elevated_temp)** 데이터 — 상세 패널에 표로 표시
- 🟡 온도별 데이터가 **표만** 제공 → 곡선 차트·다재료 오버레이 없음
- ⚠️ **크리프 / 응력-파단(stress-rupture)** 데이터 없음
- ⚠️ **열팽창계수(CTE)** 없음 — 열응력·끼워맞춤·열변형 계산에 필수
- ⚠️ 산화/내환경은 정성 등급뿐(특정 분위기·온도별 산화율 없음)

### S3. 내식 / 해양·화학 환경
특정 환경 내식성 + 강도.
- ✅ 내식성 정성 등급 필터
- ⚠️ 환경별(공식 pitting / 입계 / 응력부식균열 SCC / 갈바닉 / 매질별) 구분 없음
- ⚠️ 정성 등급만 — 정량(부식속도 mm/yr, PREN 등) 없음

### S4. 원가 제약 양산
강도/강성 만족하는 최저가 재료.
- ✅ `price_per_kg`·`price_per_cm3`, 인덱스 `E/Cm`·`σy/Cm`, Compare에 가격 포함
- 🟡 가격은 "approximate raw-material" — 지역/변동성/시점 미반영
- ⚠️ **총 부품 원가**(가공비, AM 빌드시간·buy-to-fly, 후처리) 모델 없음

### S5. 피로 임계 (회전·진동 부품)
내구한도, S-N, 노치 민감도, 평균응력 영향.
- ✅ `fatigue_strength`(R=-1 한도) 필터·비교, est. 라벨로 정직 표기
- ⚠️ **S-N 곡선 / R비 / 표면조도 감쇠 / AM 결함 산포** 없음 (단일 내구한도값)
- ⚠️ Goodman/Haigh 평균응력 보정 도구 없음

### S6. 정밀 강성 (변형·고유진동수 제어)
E, 밀도(공진), 열안정성(CTE), 감쇠.
- ✅ E·밀도, 비강성 인덱스(`E/ρ` 등 = 고유진동수 지표)
- ⚠️ **CTE / 손실계수(damping·loss coeff) / 비열 / 푸아송비** 없음
- ⚠️ 고유진동수·열변형 직접 계산 보조 없음

### S7. 접합 / 용접 / 조립
- ✅ 용접성 정성 등급 필터
- ⚠️ 이종 접합·용접후열처리(PWHT)·접합 설계 가이드 없음

### S8. 규격 / 인증 / 추적성
어느 규격·데이터시트·등급(항공/의료)인지.
- ✅ 출처(sources) 검증 URL, 조성 범위, 교차규격 이명(UNS/EN/DIN/JIS/GB/AISI)
- 🟡 열처리 조건(heat_treatment) 필드는 있으나 비교축으로 노출 약함
- ⚠️ 인증 플래그(AMS, ASTM F, ISO, 생체적합) 분류·필터 없음
- ⚠️ 단일 재료 **데이터시트(PDF) 리포트** 생성 없음

### S9. AM vs 단조/주조 (공정 선택)
같은 합금의 LPBF vs 단조 비교, 이방성·기공.
- ✅ AM·전통 공정 데이터가 같은 재료군에 포함, 공정 필터로 구분
- 🟡 명시적 "AM vs Wrought 나란히" 토글·상대밀도(porosity) 비교 약함
- ⚠️ as-built / HIP / 열처리 **조건 선택자**로 물성 비교 불가(범위에 통합됨)

### S10. 다목적 트레이드오프
강성 vs 질량 vs 원가 동시 최적.
- ✅ **다중 인덱스 제약(N개 AND)** — 이번 라운드 구현
- ✅ Ashby 산점도 + 외피(envelope) + 축 한계(limit)
- ⚠️ **3번째 물성 매핑**(점 색/크기 = 예: 가격·온도) 없음 — 색은 카테고리 고정
- ⚠️ **파레토 전선(Pareto front)·knee 강조**, 가중 점수 랭킹 없음

## 2. 개선점 정리 (우선순위)

### P0 — 분석 가치 높고 구현 현실적 (우선)
1. **차트 3번째 물성 매핑(버블)** — 점 색/크기를 임의 물성(가격·내열·밀도)에 연속 매핑. 단일 차트에서 3~4목적 트레이드오프 파악.
2. **파레토 전선 + 가중 점수 랭킹** — 선택 집합에서 비지배 후보 강조, 가중치 슬라이더로 종합 점수 정렬.
3. **온도-물성 곡선 차트** — 이미 수집된 `elevated_temp`를 σy/UTS vs T 곡선으로, 다재료 오버레이.
4. **단위계 토글(SI ↔ Imperial)** — ksi, lb/in³, ksi√in 등. 실무(특히 북미 항공) 필수.
5. **물성 정의·조건 툴팁** — 시험조건(RT, R=-1, 0.2% offset), 규격, est. 여부를 hover로. 오용 방지.
6. **AM 이방성 정량화** — XY vs Z 값이 있는 경우 분리 표시·비교(현재 텍스트 노트만).

### P1 — 데이터·기능 확장 (중요)
7. **누락 물성 추가**: CTE(열팽창), 푸아송비, 비열, 융점/고상선, 손실계수. *(CTE 최우선 — 열응력/끼워맞춤)*
8. **크리프 / 응력-파단** 데이터(고온 시나리오 S2).
9. **요구사항 → 랭킹 후보 모드** — 목표 스펙 입력 시 만족 후보를 점수순 정렬(역탐색).
10. **인증/규격 분류·필터** — AMS/ASTM F/ISO/biocompatible 태그.
11. **Compare 기준재(baseline) 정규화** — 특정 재료 대비 비율(%)로 표시.
12. **단일 재료 데이터시트(PDF) 리포트** 내보내기.
13. **피로 맥락 강화** — R비·표면조도 감쇠·AM 산포 주석(데이터 한계 명시).

### P2 — 향후/심화
14. **AM 출력성 제약** — 최소 피처, 오버행/서포트, 빌드 부피, as-built 조도, HIP/열처리 조건 선택자.
15. **총 부품 원가 모델** — 가공비 + AM 빌드시간·buy-to-fly + 후처리, 지역/변동성.
16. **환경별 정량 부식** — pitting/SCC/갈바닉, PREN, 매질·온도별.
17. **접합 가이드** — 이종 접합·PWHT.
18. **지속가능성 지표** — 내재에너지/CO₂, 재활용성.
19. **애플리케이션/용도 검색**, 선택 프로젝트 저장(컬렉션 확장).

## 3. 현재 강점 (유지)
- 범위 기반(min/max/typical/n) 데이터 + 검증 출처 — 값/인용 날조 없음, est. 정직 라벨.
- Ashby 워크플로우 완성도: 11개 인덱스, **N개 다중 제약**, box-select, 축 한계, 외피(1차 family 기본), Compare 연동 색칠.
- Compare: 최대 30종, 물성별 가로 막대 + 정렬, **드래그 폭 조절(이번 라운드)**.
- 컬렉션 저장 + 공유 URL, CSV 내보내기, 차트→필터 핀.
- 반응형(모바일 전체화면 오버레이), 775 재료(금속+폴리머).

## 4. 진행 현황 & 구현 로드맵 (천천히)

### 이미 완료 (✅)
- **P1-7 누락 물성**: CTE·푸아송비·비열·융점 추가(클래스 대표값, est. 라벨, Invar 특수처리). 필터·축·Compare·CSV·상세 노출. *(손실계수만 미반영)*
- **P0-3 온도-물성 곡선**: 상세(σy/UTS-vs-T) + Compare 다재료 오버레이(σy/UTS 토글).
- **Compare→상세 연동**: Compare에서 재료 클릭 → Ashby 차트로 전환 + 링 마커 강조 + 상세 패널 오픈.
- **Compare 폭 드래그 조절**(저장됨) · **N개 다중 인덱스 제약** · 한계선 드래그 버그 수정.

### 단계별 로드맵 (각 단계 = 독립 배포 가능 단위 · 노력 S/M/L)
> 데이터 무결성 원칙(측정값/인용 날조 금지, 추정값 라벨) 전 단계 공통 적용.

**Phase A — 분석력 (Ashby 선택론 강화)**
- A1 `M` **3번째 물성 버블 매핑** (P0-1): 점 색/크기를 임의 물성(가격·내열·CTE)에 연속 매핑 + 범례. 색상 카테고리와 토글.
- A2 `M` **파레토 전선 + 가중 랭킹** (P0-2): 선택 집합의 비지배 후보 강조, 가중치 슬라이더로 종합 점수 정렬·표시.
- A3 `M` **요구사항→랭킹 모드** (P1-9): 목표 스펙 입력 → 만족 후보를 점수순(역탐색). A2 랭킹 로직 재사용.

**Phase B — 정확한 사용·해석 (오용 방지)**
- B1 `S` **물성 정의·조건 툴팁** (P0-5): 시험조건(RT, R=-1, 0.2% offset)·규격·est. 여부 hover. (PropertyMeta.description 이미 보유 → 노출만.)
- B2 `M` **단위계 토글 SI↔Imperial** (P0-4): ksi·lb/in³·°F 등. 표시 레이어 변환(데이터는 SI 유지).
- B3 `S` **Compare 기준재 정규화** (P1-11): 선택 기준 재료 대비 비율(%).

**Phase C — AM 특화**
- C1 `M` **이방성 정량화** (P0-6): XY vs Z 값 보유 시 분리 표시·비교(현재 노트만). 데이터 모델에 방향축 추가 필요.
- C2 `L` **출력성 제약** (P2-14): 최소 피처·오버행·빌드부피·as-built 조도·HIP/열처리 조건 선택자.

**Phase D — 데이터 심화**
- D1 `M` **크리프/응력-파단** (P1-8): 고온 합금 핸드북 stress-rupture(신뢰값만, 라벨).
- D2 `S` **손실계수·추가 물성** (P1-7 잔여) + **피로 맥락 주석** (P1-13: R비·표면조도·AM 산포 한계 명시).
- D3 `M` **인증/규격 태그** (P1-10): AMS/ASTM F/ISO/biocompatible 분류·필터.

**Phase E — 산출물·확장**
- E1 `M` **단일 재료 데이터시트 PDF** (P1-12).
- E2 `L` **총 부품 원가 모델** (P2-15) · **환경별 정량 부식**(P2-16) · **지속가능성 지표**(P2-18) — 데이터 확보 선행.

### 추천 순서
A1 → A2 → B1 → B2 → A3 → C1 → D2 → (이후 D1/D3/E 단계는 데이터 확보 상황 따라).
근거: 분석력(A)·해석 안전성(B)이 사용 가치가 가장 크고 기존 구조 재사용도가 높음. C·D·E는 데이터 수집이 선행되어야 하므로 병행하며 천천히.
